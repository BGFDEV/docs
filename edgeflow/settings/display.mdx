Display settings control how EdgeFlow appears to researchers and participants. Configure these to optimize for your environment and accessibility needs.

## Accessing Settings

Navigate to **Settings** > **Display** to view and modify these options.

---

## Participant Window

### Window Mode

| Option | Description |
| --- | --- |
| Fullscreen | Participant window covers entire screen (recommended) |
| Windowed | Participant window in resizable window |

<Tip>
Fullscreen mode removes distractions and prevents accidental closure. Use windowed mode only for development or troubleshooting.
</Tip>

### Stimulus Display

| Setting | Default | Range | Description |
| --- | --- | --- | --- |
| Display Size | 75% | 25-100% | Stimulus size relative to screen |
| Padding | 10% | 0-25% | Space around stimulus edges |
| Background Color | Black | Color picker | Background during assessment |

---

## Researcher Window

### Window Behavior

| Setting | Default | Description |
| --- | --- | --- |
| Default Size | Normal | Window size on launch |
| Remember Position | On | Restore window position on restart |
| Always on Top | Off | Keep window above other applications |

---

## Theme

### Color Mode

| Option | Description |
| --- | --- |
| Light | Light background with dark text |
| Dark | Dark background with light text |
| System | Follow system preference |

### High Contrast Mode

Enable high contrast for improved visibility:

| Setting | Default | Description |
| --- | --- | --- |
| High Contrast | Off | Increase color contrast |
| Reduce Motion | Off | Minimize animations |

---

## Typography

### Font Size

| Option | Scale |
| --- | --- |
| Small | 90% |
| Medium (default) | 100% |
| Large | 110% |
| Extra Large | 125% |

Font size affects all text in the researcher interface, including buttons, labels, and data tables.

<Note>
Participant window text size is configured separately to ensure optimal readability during assessments.
</Note>

### Participant Instructions

| Setting | Default | Description |
| --- | --- | --- |
| Instruction Font Size | Large | Text size for participant instructions |
| Instruction Duration | 5 seconds | How long instructions display |

---

## Accessibility

### Visual Accommodations

| Setting | Default | Description |
| --- | --- | --- |
| High Contrast Mode | Off | Increase contrast throughout UI |
| Large Cursor | Off | Enlarge cursor for visibility |
| Focus Indicators | On | Show visible focus outlines |

### Motion

| Setting | Default | Description |
| --- | --- | --- |
| Reduce Motion | Off | Minimize transitions and animations |
| Reduce Transparency | Off | Make backgrounds fully opaque |

<Tip>
Enable these options based on researcher and participant needs. Reduced motion can also improve performance on older hardware.
</Tip>

## Multi-Display Setup

EdgeFlow supports multi-display configurations:

### Recommended Setup

1. **Primary Display**: Researcher monitor showing control interface
2. **Secondary Display**: Participant window in fullscreen

### Configuring Displays

<Steps>
  <Step title="Connect Second Monitor">
    Connect and configure your secondary display at the OS level.
  </Step>
  <Step title="Launch EdgeFlow">
    Open EdgeFlow on your primary display.
  </Step>
  <Step title="Start a Session">
    When you launch a session, the participant window opens.
  </Step>
  <Step title="Move to Secondary Display">
    Drag the participant window to the secondary display.
  </Step>
  <Step title="Enable Fullscreen">
    The window position is remembered for future sessions.
  </Step>
</Steps>

### Display Assignment

| Setting | Description |
| --- | --- |
| Default Participant Display | Which monitor opens participant window |
| Remember Position | Save participant window location |

## Preview Mode

Test your display settings without running a full session:

<Steps>
  <Step title="Open Settings">
    Navigate to **Settings** > **Display**.
  </Step>
  <Step title="Click Preview">
    Click **Preview Participant Window**.
  </Step>
  <Step title="Review Display">
    A sample participant screen appears with your current settings.
  </Step>
  <Step title="Adjust and Re-preview">
    Modify settings and preview again until satisfied.
  </Step>
</Steps>

---

## Resetting to Defaults

To restore all display settings:

1. Navigate to **Settings** > **Display**
2. Click **Reset to Defaults**
3. Confirm the reset

<Warning>
This resets all display settings, including window positions.
</Warning>
