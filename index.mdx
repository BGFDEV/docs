## Key Features

<CardGroup cols={3}>
  <Card title="VMSWO Assessments" icon="list-check">
    Structured verbal preference assessments with automatic counterbalancing
  </Card>
  <Card title="Conjugate Reinforcement" icon="hand-fist">
    Force-based behavioral engagement measurement
  </Card>
  <Card title="Correspondence Analysis" icon="chart-scatter">
    Compare verbal and behavioral preference measures
  </Card>
  <Card title="Encrypted Storage" icon="lock">
    Secure local data storage with SQLCipher encryption
  </Card>
  <Card title="Data Export" icon="file-export">
    Export to CSV or JSON for external analysis
  </Card>
  <Card title="Cross-Platform" icon="desktop">
    Runs on macOS, Windows, and Linux

</CardGroup>
